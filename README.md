 RideFlow

RideFlow is a web-based ride management system built with Node.js, Express, and SQL.
It provides separate dashboards for riders, drivers, and admins, enabling ride requests, management, and reporting in a structured workflow.

- Features
 User authentication (Riders & Drivers)
 Ride request system
 Driver dashboard for managing rides
 Admin panel for monitoring system activity
 Reports and analytics dashboard
 SQL-based database integration
 EJS templating for server-side rendering
 
- Tech Stack
Backend: Node.js, Express.js
Frontend: HTML, CSS, EJS
Database: SQL (MySQL),cloud database Aiven
Middleware: Express middleware (auth system included)
